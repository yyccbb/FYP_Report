\section{Plan for Next Semester}
The plan for the next semester is outlined as follows:
\begin{enumerate}
    \item Refactor the existing TensorFlow-based codebase to PyTorch for improved compatibility and flexibility, and reorganize the current project structure for better maintainability.
    \item Adapt the GiGPO-style reward calculation to the TSC context. Test and make necessary adjustments to the formulation and give it a cool name.
    \item Implement the asynchronous message propagation mechanism as described in the \textit{Traffic-R1} paper.
    \item Conduct comprehensive experiments and benchmark the proposed framework against state-of-the-art (SOTA) baselines.
    \item Prepare and finalize the report documenting the methodology, results, and findings.
\end{enumerate}
